Software I use

This is the start of a para. Lots of really cool programs can be found on the internet and not all of them are impossible to understand or require dark fucking magic to grok. Not every piece of software is metamorphic malware written by god. Some of it though, does require insane amounts to work to take apart and study the machine code. That is rarely the case, by the way. Not everyone wants to audit their own firmware, for example. That'd be very impressive if one can, reverse engineers and binary analysts with decades of experience pull it off like it's one of their daily chores, which is unironcally also true.

However, some computer literacy is always good. Not fancy algorithms, just the bare fundamentals that make the foundation for the modern bloated towers of abstract that we're so used to, though majority of it we don't really need.   • Operating systems: Windows is fine. Learn how hypervisors work and microsoft "officially" lets you run linux VMs in hyper-V (their virtualization stack) when you need them.
  • Text editors: Hate me for me but I'm an Emacs fan, I think it's great software.
  • Email clients: Anything that lets you read and write plain ASCII text emails is fine. I personally like Mutt and Claws Mail.
  • PDF readers: Zathura and Okular are nice.
  • Standalone sandboxes: Firejail and Sandboxie-plus.
  • Compiler and binary-hacking suite: LLVM
  • Browser: Firefox
